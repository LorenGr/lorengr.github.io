Overview

AngularFlatStrap is an evolving collection of components that can be used within your AngularJS application.

Dependencies

  • AngularJS (requires AngularJS 1.3)
  • FlatStrap CSS ('Flat' design theme)

Files to download

Build files for all directives are distributed in a main single file called build/angular-flatstrap.js (minified for production usage). This file contains all the directives and templates you will need to impliment the components in your app. If you only wish to include individual components, you can find the build files in the folder /src/[component name]/build/[component name].min.js

Installation

Install AngularFlatStrap

As soon as you've got all the files downloaded and included in your page you just need to declare a dependency on the flatstrap module:

{% highlight text %} angular.module("myModule",["flatstrap"]) {% endhighlight %}

Bower Install

{% highlight text %}$ bower install angular-flatstrap{% endhighlight %}
